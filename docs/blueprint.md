# **App Name**: ChatterBox

## Core Features:

- Real-time Messaging: Send and receive text messages in real-time.
- Group Chat: Participate in group conversations within specified rooms.
- Private Messaging: Engage in one-on-one private conversations with other users.
- Delivery Status: Indicate whether messages have been sent, delivered, and read.
- Typing Indicators: Show when a user is currently typing a message.
- Intelligent Response Suggestions: Suggest contextually relevant responses using a generative AI tool to help users quickly reply.
- File Sharing: Allow users to upload and share files within chats.

## Style Guidelines:

- Primary color: Light blue (#A7D1F7) to create a calm and trustworthy environment.
- Background color: Very light blue (#F2F8FD) to ensure readability and a clean aesthetic.
- Accent color: Soft Green (#B1F2B3) for sent message confirmation and highlights.
- Font pairing: 'Inter' sans-serif for both headline and body text, providing a modern, clean, and easily readable interface.
- Simple, outlined icons for navigation and actions to maintain a minimalist aesthetic.
- Clean and intuitive layout with clear separation of messages and input areas for easy navigation.
- Subtle animations for message delivery and read confirmations to provide visual feedback without being intrusive.
